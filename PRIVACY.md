  # Privacy Policy for Tiny URL EmBiginator
  Effective date: March 5, 2026

  Tiny URL EmBiginator processes website content only to provide its core feature: resolving user-submitted short/wrapped URLs to their final destination.

  ## Data We Process
  - URLs entered by the user in the extension popup.
  - Redirect chain data (intermediate URLs, status codes, final URL).
  - For some LinkedIn short links, page HTML needed to extract the outbound destination.
  - Optional domain health lookup data (queried domain and returned metadata).

  ## How We Use Data
  - To show users where a short link leads.
  - To display redirect hops and final domain comparison.
  - To provide optional domain-health results.
  - To keep a local lookup history for user convenience.

  ## Storage
  - Lookup history is stored locally on the user’s device using Chrome extension local storage.
  - We do not run a developer backend to store this data remotely.

  ## Sharing
  - URL/domain data is sent only as required to perform lookups:
    - Requests to the entered URL and redirect targets.
    - Optional requests to `crt.sh` and `rdap.org` when Domain Health is used.
  - We do not sell user data.

  ## Limited Use
  We use website content solely for user-facing functionality described in the Chrome Web Store listing and extension UI.
